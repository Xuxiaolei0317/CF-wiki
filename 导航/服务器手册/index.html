    系统和软件版本查看命令¶

    查看系统版本¶

    cat /etc/issue

    查看CentOS系统版本¶

    cat  /etc/redhat-release

    查看内核版本¶

    uname -a

    下载数据¶

    curl指令（也可以下载其他后缀的文件）

    curl -O https://www.example.com/foo/bar.html

    参考：https://www.ruanyifeng.com/blog/2019/09/curl-reference.html

    常用指令¶

    创建软链接¶

    ln -s指令 下面代码会将为python3.8创建一个软链接 

    sudo ln -s /usr/bin/python3.8 /usr/bin.python

    常用设置¶

    pip换源¶

    cd ~
mkdir ~/.pip
cd ~/.pip
sudo vim ~/.pip/pip.conf
    在pip.conf中写入如下内容：

    [global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple/ 
[install]
trusted-host = pypi.tuna.tsinghua.edu.cn

    conda设置¶

    1. 安装miniconda¶

    curl -O https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/Miniconda3-py38_4.9.2-Linux-x86_64.sh
sh Miniconda3-py38_4.9.2-Linux-x86_64.sh

    需要配置Python的环境变量，用vim打开vim ~/.bashrc。 在尾部添加export PATH=/opt/miniconda3/bin:$PATH，保存退出。

    source ~/.bashrc

    1. 创建新环境¶

    conda create -n <env_name> python=X.X e.g.conda create -n py36 python=3.6

    2. 拷贝旧环境¶

    创建一个新环境，拷贝已有环境old： 

    conda create -n new --clone old

    2. conda换源¶

    linux将~/.condarc改为以下的内容： 

    channels:
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/conda-forge/
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/Paddle/
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/fastai/
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/bioconda/
show_channel_urls: true
ssl_verify: false
