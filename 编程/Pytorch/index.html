    基础操作

    在指定维度上将tensor复制

    • Tensor.repeat(*sizes)
    • 注意和np.repeat不同，

    例如：

    === "Python"

    ```python
x = torch.tensor([1, 2, 3])
x.repeat(4, 2)
# tensor([[ 1,  2,  3,  1,  2,  3],
#         [ 1,  2,  3,  1,  2,  3],
#         [ 1,  2,  3,  1,  2,  3],
#         [ 1,  2,  3,  1,  2,  3]])
```

    指定维度用该维度上的索引来选择元素

    • torch.gather(input, dim, index, *, sparse_grad=False, out=None)

    dim表明我们确定了某个维度，索引矩阵是该维度上的索引，其他索引间是不会有交互的。

    假设输入矩阵为2维： - 根据dim=0选择元素：out[i][j] = input[index[i][j]][j]

    则我们可以得到默认的index矩阵，torch.gather(input,0,index) 还是等于input.

    [[0 0 0],
 [1 1 1],
 [2 2 2]]

    t = torch.tensor([[1, 2], [3, 4]])
torch.gather(t, 0, torch.tensor([[0, 0], [1, 0]]))
# [[1 2], [3 2]]

    模型创建

    nn.Sequential()手动添加module

    nn.Sequential(*modules)中的Module会按照str(位置索引)作为name.

    model = nn.Sequential(
          nn.Conv2d(1,20,5),
          nn.ReLU(),
          nn.Conv2d(20,64,5),
          nn.ReLU()
        )
print(list(model.modules()))
# -------- 上下两种代码功能相同。
model1 = nn.Sequential()
model1.add_module(str(len(model1)), nn.Conv2d(1,20,5))
model1.add_module(str(len(model1)), nn.ReLU())
model1.add_module(str(len(model1)), nn.Conv2d(20,64,5))
model1.add_module(str(len(model1)), nn.ReLU())

print(list(model1.modules()))

    Tensorboard指南

    代码模板

    from torch.utils.tensorboard import SummaryWriter
import numpy as np

writer = SummaryWriter()

for n_iter in range(100):
    writer.add_scalar('Loss/train', np.random.random(), n_iter)
    writer.add_scalar('Loss/test', np.random.random(), n_iter)
    writer.add_scalar('Accuracy/train', np.random.random(), n_iter)
    writer.add_scalar('Accuracy/test', np.random.random(), n_iter)

    对于模型一次训练中一个Step需要输出多个结果，可以使用add_scalars
