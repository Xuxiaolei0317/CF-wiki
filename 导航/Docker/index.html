基础操作

容器后台运行退出

Ctrl + p + q

后台容器重新进入命令

docker exec -it [container id] /bin/bash

暂停正在运行的container

docker stop [CONTAINER ID]

删除container

container的STATUS非UP状态时可以删除，否则需要先暂停。

docker ps -a # 查看所有的container
docker rm [CONTAINER ID]

主机和容器传输文件

# 主机传到容器
docker cp [host_path] [containerID]:[container_path]
# 容器到主机
docker cp [containerID]:[container_path] [host_path]

配置服务器上Docker的tensorflow-gpu

下载tensorflow-gpu的image

前往https://hub.docker.com/上搜索想要的image版本。（我这里选择的是TAG=1.15.0-gpu-py3的版本）

docker pull tensorflow/tensorflow:1.15.0-gpu-py3

创建Tensorflow-gpu的container

sudo nvidia-docker run -p 5592:5592 -p 5593:5593 -p 8022:22 --name="tf" -v ~/tf:/home/xuyc/workspace/tf -it tensorflow/tensorflow:1.15.0-gpu-py3 /bin/bash

若出现了invalid volume specification错误，可参考下面的问题解答。

配置SSH服务

apt update
apt install -y openssh-server

mkdir /var/run/sshd
echo 'root:密码' | chpasswd
sed -i 's/PermitRootLogin prohibit-password/PermitRootLogin yes/' /etc/ssh/sshd_config
sed 's@session\s*required\s*pam_loginuid.so@session optional pam_loginuid.so@g' -i /etc/pam.d/sshd
echo "export VISIBLE=now" >> /etc/profile
echo "PermitRootLogin yes" >> /etc/ssh/sshd_config #额外添加的

重启SSH激活配置：

service ssh restart

在服务器（宿主机）上（不是服务器的docker里）测试刚刚新建docker容器中哪个端口转发到了服务器的22端口：

sudo docker port [your_container_name] 22
# 如果前面的配置生效了，你会看到如下输出
# 0.0.0.0:8022

sudo nvidia-docker run -p 5592:5592 -p 5593:5593 -p 8022:22 --name="tf" -v ~/tf:~/Desktop/Projects/MyResearch/tf -it tensorflow/tensorflow:1.15.0-gpu-py3 /bin/bash

保存镜像 docker commit -m "xxx" [CONTAINER ID]

添加REPOSITORY和TAG信息 docker tag [IMAGE ID] reposistory:tag docker tag [IMAGE ID] tensorflow:1.15

查看容器挂载映射目录 docker inspect [CONTAINER ID] | grep Mounts -A 20

主机和容器拷贝文件（在主机中操作） 容器->主机：docker cp [CONTAINER NAME]:[path] [path] 主机->容器：docker cp [path] [CONTAINER NAME]:[path]

这里报了一个错，不可以使用

docker: Error response from daemon: invalid volume specification: '/home/xuyc/tf:~/Desktop/Projects/MyResearch/tf': invalid mount config for type "bind": invalid mount path: '~/Desktop/Projects/MyResearch/tf' mount path must be absolute.
