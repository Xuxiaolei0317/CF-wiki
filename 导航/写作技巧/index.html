    Markdown技巧

    Markdown适配知乎文章编辑器

    知乎支持Markdown文件导入，但无法识别tex公式。通过以下的方式，可以将$...$和$$...$$替换为知乎支持的公式。

    import sys
import re

interline_tag = '\n<img src="https://www.zhihu.com/equation?tex={}" alt="{}\\\\" class="ee_img tr_noresize" eeimg="1">\n'
interline_pattern = "\$\$\n*(.*?)\n*\$\$"
inline_tag = '<img src="https://www.zhihu.com/equation?tex={}" alt="{}" class="ee_img tr_noresize" eeimg="1">'
inline_pattern = "\$\n*(.*?)\n*\$"

def replace_tex(content):
    def dashrepl(matchobj, tag):
        formular = matchobj.group(1)
        return tag.format(formular, formular)

    content = re.sub(interline_pattern, lambda mo: dashrepl(mo, interline_tag), content)
    content = re.sub(inline_pattern, lambda mo: dashrepl(mo, inline_tag), content)

    return content

if __name__=='__main__':
    assert len(sys.argv) > 1, "Error: need filename as a argument"
    filename = sys.argv[1]
    with open(filename, 'r') as f:
        content = f.read()
    with open(filename, 'w') as f:
        f.write(replace_tex(content))

    基础概念和设置

    基础概念

    • 排版工具：pdfTeX, pdfLaTeX, XeTeX, XeLaTeX 等
    • LaTeX将源文件中的换行当作是空格；需要另起一段，则需要用两个换行（一个空行）实现。

    使用中文

    使用ctexart文档类。

    \documentclass[UTF8]{ctexart}
\usepackage[T1]{fontenc} %防止某些符号显示不正确

    或使用ctex宏包，推荐使用xelatex编译。

    \documentclass{article}
\usepackage[UTF8]{ctex}

    参考：1. 全面总结如何在 LaTeX 中使用中文 (2020 最新版)

    插入图像

    最好使用graphicx环境提供的\includegraphics命令。

    最简单的用法

    \documentclass{article}
\usepackage{graphicx}
\begin{document}

\begin{figure}
\centering % 插图居中
\includegraphics{a.jpg}
\caption{有图有真相}
\label{fig:myphoto}
\end{figure}

\end{document}

    控制图像的大小（推荐）

    \includegraphics[width = .8\textwidth]{a.jpg} - .8\textwidth：表示页面宽度的百分之八十。

    浮动体

    图像和表格通常需要占据大块空间，需要对figure和table环境进行设置，让其自动调整位置。

    • htb分别表示here, top, bottom，以h-t-b的顺序，使得latex尽量满足排在前面的浮动格式
    • 还可以使用float宏包的H选项，强制放在当前段落后。放不下的话，需要手动调整。
    \begin{figure}[htb]
\centering
\includegraphics{a.jpg}
\caption{有图有真相}
\label{fig:myphoto}
\end{figure}

    参考： 1. LaTeX 中的浮动体：基础篇

    插入数学公式

    需要加载amsmath宏包。二元关系符：加载amssymb宏包

    数学模式

    • 对行间公式编号，使用equation环境，equation*则是默认不编号。
    • 多行公式对齐使用align环境，莫用eqnarray

    basicstyle=\ttfamily\footnotesize,
