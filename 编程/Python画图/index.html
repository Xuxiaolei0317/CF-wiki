基本介绍

Figure和Axes

matplotlib中最基本的对象是Figure和Axes. - Axes：坐标系，或者可以理解为绘图的最小显示区域。必须依附Figure存在。 - Figure就是一个插图（类比：一个窗体或Jupyter中的部件），一个Figure可以包含至少一个Axes，既然是插图，就可以放多个数据图像、坐标系进行比较。

接下来，先简单地创建一个Figure和一个Axes对象，并进行绘图。 - python fig, ax = plt.subplots() # Create a figure containing a single axes. ax.plot([1, 2, 3, 4], [1, 4, 2, 3]) # Plot some data on the axes.

Figure的构成

解释： - Axes：坐标系，包含Axis坐标轴，title标题，label坐标轴的标签。 - Axis：坐标轴，二维坐标系有二个坐标轴，三维坐标系（三个）。 可通过Axes.set_xlim()或Axes.set_ylim()限制显示的范围。

2. 到底如何使用plot函数，ax.plot还是plt.plot？

其实对应的是两种使用matplotlib的风格。 - 面向对象风格：需要显式地创建Figures和Axes对象，并调用他们的方法，例如ax.plot（ax是Axes类的一个实例化） - pyplot接口风格：依靠pyplot自动地创建和管理Figures和Axes，并使用pyplot函数进行绘图。（厉害了啊） - 对比如下： - 面向对象 python x = np.linspace(0, 2, 100) # Note that even in the OO-style, we use `.pyplot.figure` to create the figure. fig, ax = plt.subplots() # Create a figure and an axes. ax.plot(x, x, label='linear') # Plot some data on the axes. ax.plot(x, x**2, label='quadratic') # Plot more data on the axes... ax.plot(x, x**3, label='cubic') # ... and some more. ax.set_xlabel('x label') # Add an x-label to the axes. ax.set_ylabel('y label') # Add a y-label to the axes. ax.set_title("Simple Plot") # Add a title to the axes. ax.legend() # Add a legend. - pyplot接口 python x = np.linspace(0, 2, 100) plt.plot(x, x, label='linear') # Plot some data on the (implicit) axes. plt.plot(x, x**2, label='quadratic') # etc. plt.plot(x, x**3, label='cubic') plt.xlabel('x label') plt.ylabel('y label') plt.title("Simple Plot") plt.legend() 几乎没有什么差别。

3. 绘制表格图

要求：每个坐标轴上的相邻刻度围成的格子是正方形的。

import matplotlib.pyplot as plt
"""
绘制表格图
"""
plt.figure(figsize=(width, height)) # 创建一个Figure对象
plt.grid() # 显示表格
plt.xticks([i for i in range(0, width + 1)]) # 设置x轴刻度
plt.yticks([i for i in range(1, height + 1)]) # 设置y轴刻度
axes = plt.gca() # 得到当前的Axes对象
plt.show()

4. 绘制箭头

需求：画出一个长度为1的箭头。

import matplotlib.patches as mpatches
import matplotlib.pyplot as plt
actions = [(0, 1), (0, -1), (-1, 0), (1, 0)]  # 上，下，左，右
action = 2 # 表示箭头朝着左侧，长度为1.
# i,j是箭头尾部的横坐标和纵坐标。
tail = (i + 0.5, j + 0.5)
head = (tail[0] + actions[action][0] / 2, tail[1] + actions[action][1] / 2)
arrow = mpatches.FancyArrowPatch(tail, head, mutation_scale=10)
axes.add_patch(arrow)

plt.show()

7. 一个Ax显示多个函数的对比图

import matplotlib.pyplot as plt
from cycler import cycler
import matplotlib as mpl

# 绘制2各函数图像
compare_list = ["y=x+1", "y=-2x+10", 
               ]
x = list(range(10))
data_list = [[i + 1 for i in x],
             [-2*i + 10 for i in x]]

colors = ['#1f77b4', '#ff7f0e', '#2ca02c', '#d62728', '#9467bd', '#8c564b', '#e377c2', '#7f7f7f', '#bcbd22', '#17becf', '#1a55FF']

colors = [colors[i] for i in range(len(compare_list))]

mpl.rcParams['axes.prop_cycle'] = cycler(markevery=[1,1], color=colors)
fig, ax = plt.subplots()
name = "value"
for i in range(len(compare_list)):
    ax.plot(data_list[i], marker='o', label=str(compare_list[i]))
    ax.legend(loc="best")
    ax.set_xlabel('epochs',fontsize=12)
    ax.set_ylabel(name,fontsize=12)
    ax.set_xticks(x)
plt.show()
