变量使用

定义变量 - var=1 - 注意：=周围不能有空格。 - 字符串单引号和双引号用法不同，通常使用双引号

使用变量

var = 1
echo $var
echo ${var} # 推荐使用变量的时候加上花括号{}

修改变量的值

var = 1
var = 2 # 不需要加$

单引号和双引号的区别 - 单引号会将引号中的内容不解析直接输出。 - 双引号能够解析字符串中变量和命令，再输出。

数组

定义数组：数组名=(值1 值2 ... 值n) 读取数组：${数组名[下标]}, @可以获得数组中所有元素：${array_name[@]}

if判断（数字条件和字符串条件）

语句：

if [ $a -lt $i ]
then
    echo "a < i"
else
    echo "a > i"
fi

整数比较 |符号|含义|做法| | ---- | ---- | ---- | |-eq | 等于 |if [ "$a" -eq "$b" ] | |-ne | 不等于 | if [ "$a" -ne "$b" ] | |-gt | 大于 | if [ "$a" -gt "$b" ] | |-ge | 大于等于 | if [ "$a" -ge "$b" ] | |-lt | 小于 | if [ "$a" -lt "$b" ] | |-le | 小于等于 | if [ "$a" -le "$b" ] | |< | 小于(需要双括号) | (("$a" < "$b")) | |<= | 小于等于(需要双括号) | (("$a" <= "$b")) | |> | 大于(需要双括号) | (("$a" > "$b")) | |>= | 大于等于(需要双括号) | (("$a" >= "$b")) |

字符串比较 |符号|含义|做法| | ---- | ---- | ---- | |= |等于 | if [ "$a" = "$b" ] | |== |等于,与=等价 | if [ "$a" == "$b" ] |
